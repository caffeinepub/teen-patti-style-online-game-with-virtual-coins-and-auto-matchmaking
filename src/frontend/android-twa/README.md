# Android TWA (Trusted Web Activity) Packaging Guide

This directory contains the configuration and instructions for packaging the Teen Patti web app as an Android application using Trusted Web Activity (TWA).

## Prerequisites

- Android Studio (latest stable version)
- Java Development Kit (JDK) 11 or higher
- Node.js and npm (for building the web app)
- Your web app deployed and accessible via HTTPS
